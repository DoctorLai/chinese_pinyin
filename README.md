# chinese_pinyin
Node JS Library to Convert Chinese to Pinyin

## Installation
```
npm install chinese_pinyin
```

## Usage
```
var pinyin = require('chinese_pinyin');
console.log(pinyin("你好，我是 @justyy"));
```

Output:
```
"ni hao ，wo shi @justyy"
```
